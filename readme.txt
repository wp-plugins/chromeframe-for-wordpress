Stops certain versions of Internet Explorer from viewing your site. You can pick which versions. Displays a modal notice to those users.

== Description ==

Tired of hacking your site to work with old versions of IE? Just say no. ChromeFrame-For-Wordpress blocks old versions of IE. Pick which versions to block!

== Screenshots ==

http://sardiusgroup.com/wp-content/uploads/2009/12/screenshot-of-chromeframe-for-wordpress.png

== Installation ==

Just download and activate. This plugin cannot be configured.
